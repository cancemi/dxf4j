# Changelog

## v0.1.0 - 2026-03-01

First release.

- DXF reader and writer with lossless round-trip
- Template engine for placeholder text replacement
- Image insertion: replace TEXT entities with IMAGE entities
- Handle allocator for $HANDSEED management
- Support for DICTIONARY, IMAGEDEF, IMAGEDEF_REACTOR objects
- Minimal test suite with 14 tests
